
Clone project:   
    git clone https://github.com/Fitty87/Example

Create virtual enviroment:   
    py -m venv virt

Activate virtual enviroment:   
    cd virt\scripts\
    activate

Install requirements:
    cd ..
    cd ..
    py -m pip install -r requirements.txt

Settings Flask:
    set FLASK_APP=app.py
    set FLASK_DEBUG=1

Start Server:
    Flask run

Use localhost:
    http://127.0.0.1:5000/
